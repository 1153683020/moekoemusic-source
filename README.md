## 介绍

这是一个为[moekoemusic](https://github.com/iAJue/MoeKoeMusic)制作**DLNA**功能的用户，仓库用来存放&迭代代码

## 更新说明

**v1，v2**：制作基础功能，实现基本的播放控制

**v3**：同步软件音质选择，保留原始样式，与软件统一，同步音量控制，添加缓存设备的功能，处理“**无版权**”歌曲，统一请求发送

**v4**：除去老旧过时的`upnp-mediarenderer-client`包和未使用的包`upnp-client`，改用较新的`upnp-client-ts`，复用软件UA

**v5**：鉴于当前条件难以实现`seek`（即拖动进度）功能，我们将在该版本完全移除该功能，待条件允许或技术难题攻克以后重新尝试实现。后续测试发现，新包与原有的大部分逻辑**全部冲突**并且无法简单修改，决定**返回**使用旧的包执行所有功能；还有尝试多种方法，最终**未能解决关闭软件重新打开以后播放进度不同步的问题**，决定放弃。待后续难题攻克以后尝试重新适配。新增了自动添加防火墙例外的功能，防止因拦截导致无法控制设备。经过测试，该版本已知的功能全部正常运行。
